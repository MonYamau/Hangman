# Hangman
Игра "Виселица" - классическая словесная игра-головоломка для двух и более игроков. Цель — угадать загаданное слово, называя буквы одну за другой.
Игра "Hangman" реализована с консольным интерфейсом в процедурном стиле на Java 21.

## Инструкция
1. Запустите приложение;
2. Введите в консоль букву кириллического алфавита (допустимы как строчные, так и заглавные): "Н" - для запуска игры, "В" - для выхода из приложения;
3. При запуске игры последовательно вводите буквы кириллического алфавита (допустимы как строчные, так и заглавные);
4. Отслеживайте прогресс игры в отображаемом интерфейсе;
5. По окончанию игры введите в консоль букву кириллического алфавита (допустимы как строчные, так и заглавные): "Н" - для запуска игры, "В" - для выхода из приложения.
